# etao
```
1.主页
	特效：回到顶部、吸顶、轮播、选项卡、动画、间歇模型、自定义滚动条
	数据：表单序列化、underscore、js、瀑布流、懒加载、预加载、数据缓存
	功能: 查看分类、查看商品、搜索商品、加入购物车、订单查看
2.商品分类页
	特效：回到顶部、轮播、吸顶、选项卡、动画、间歇模型、自定义滚动条
	数据：表单序列化、underscore、js、瀑布流、懒加载、预加载、数据缓存
	功能: 查看分类、查看商品、搜索商品、加入购物车、订单查看
3.商品详情页
	特效：商品图片和放大镜、商品介绍、单价、总价、数量的修改等
	功能：商品详情查看、加入购物车
4.登录页（登录注册时密码传送加密后的md5值，确保传输中的安全）
	特效：实现模态框
	数据：表单序列化、模板
	功能：登录
5.注册页
	特效：实现模态框
	数据：表单序列化、模板
	功能：注册
6.购物车页面
	特效：实现商品单价和总价，添加、删除、全选、修改商品数量（最低一件，最多十件）等
	功能: 查看分类、查看商品、订单查看
7.我的订单页面
	特效：实现提交订单、取消订单等
	功能: 订单查看
8.地址页面
	特效：实现城市级联选择、默认地址、添加、删除、修改等
	功能: 查看地址、联系人、联系电话
```
